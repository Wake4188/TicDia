## Data Safety Section

### Data Collection
**Collected:**
- Email addresses (for authentication)
- Reading history (for personalization)
- App interactions (for analytics)

**Not Collected:**
- Location data
- Personal information beyond email
- Financial information

### Data Usage
- **Personalization:** Customize your feed based on reading preferences
- **Analytics:** Improve app performance and user experience
- **Account:** Manage your profile and saved articles

### Data Security
- All data encrypted in transit (HTTPS/TLS)
- Data stored securely in Supabase (SOC 2 Type II certified)
- Users can request data deletion anytime
- No data sold to third parties
