# Play Store Graphics Placeholder

Create these assets for your Play Store listing:

## Required

### Feature Graphic
- **File:** feature-graphic.png
- **Size:** 1024x500px
- **Content:** App name + tagline + screenshot mockup

### App Icon
- **File:** icon-512.png
- **Size:** 512x512px
- **Format:** PNG (32-bit, no alpha)
- **Auto-generated by:** `node scripts/generate-android-icons.js`

### Screenshots (Phone)
Minimum 2, recommended 4-8:
- screenshot-1-feed.png
- screenshot-2-article.png
- screenshot-3-categories.png
- screenshot-4-profile.png
- etc.

**Size:** 1080x1920px (9:16 aspect ratio)

## How to Create

1. **Screenshots:** Run app in Android Studio, press Cmd/Ctrl+S
2. **Feature Graphic:** Use Figma/Canva to create promotional image
3. **Icon:** Auto-generated when you run icon generation script

See [../README.md](../README.md) for detailed instructions.
